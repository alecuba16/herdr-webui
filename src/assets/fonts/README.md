JetBrainsMonoNerdFontMono-Regular.ttf is bundled from Nerd Fonts:

https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/JetBrainsMono/Ligatures/Regular

The font combines JetBrains Mono with Nerd Fonts glyphs. License files are kept next to the asset:

- JetBrainsMono-OFL.txt: SIL Open Font License 1.1 for JetBrains Mono.
